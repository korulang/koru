# Koru - The Event-Continuation Language

Koru is an event-continuation language designed for high-performance systems and metacircular compilation.

For documentation, tutorials, and more information, please visit [korulang.org](https://korulang.org).

## Repository Contents
- `src/`: Compiler source code
- `lib/`: Core system components
- `tests/regression/`: Verified examples and comprehensive technical documentation
- `koru_std/`: Standard library modules
- `run_regression.sh`: Developer utility for running the test suite
